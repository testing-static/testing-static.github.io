Preconditions

Basic Flow

Step

User Actions

System Actions

1

User login dengan menggunakan CAS menginputkan username dan password.

Sistem mengautentifikasi yang sudah diinputkan oleh user.

2

User mengklik button login.

Sistem menampilkan menu sesuai dengan akses level.

Alternate Flow

Step

User Actions

System Actions

1

Exception Flow

Step

User Actions

System Actions

1

Username dan password tidak valid.

Sistem memberikan peringatan berupa pop-up bahwa username dan password tidak sesuai.

Post conditions

Sistem memberikan konfirmasi keberhasilan login dengan sebuah message box, kemudian menampilkan form utama yang berisi tombol untuk menuju ke form lain sesuai hak akses.

Kosong

Untuk login hanya bisa diakses oleh user pada unit yang ada di UMS saja.

2. UC-002¶

F. User Guide¶

F.1. Pejabat¶

1. Login¶

  1. Masuk kedalam laman www.sarpras.ums.ac.id

  2. Pilih menu login.

  3. Login dengan mengisikan username, password, kode kemanan (gatcha).

  4. Klik tombol login.

  1. Jika username, password, serta kode keamanan benar diisikan maka, akan lanjut ke dalam menu homebase ASET.
